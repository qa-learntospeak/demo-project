# Demo Project

A sample project used for testing the [LearnToSpeak.ai](https://learntospeak.ai) GitHub MCP connector.

## About

This repository contains example code and issues to demonstrate how AI assistants (Claude, ChatGPT) can interact with GitHub repositories through the LearnToSpeak.ai MCP platform.

## Features

- User authentication with JWT
- REST API with rate limiting
- PostgreSQL database integration
- Docker deployment support

## Getting Started

```bash
npm install
npm run dev
```
